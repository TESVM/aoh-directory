Put your real mobile art files here:

- `branding/mobile/app-icon-1024.png`
- `branding/mobile/splash-2732.png`

Rules:

- icon should be a square PNG, 1024 x 1024
- splash should be a square PNG, 2732 x 2732
- keep the important logo/text in the center

After both files are here, run:

```bash
./scripts/install-mobile-assets.sh
```

That script will copy the icon and splash image into the iPhone and Android native app folders.
